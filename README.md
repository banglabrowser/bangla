# Bangla Browser

**Privacy-respecting. Bengali-first. Heritage-powered.**

Bangla Browser is a desktop web browser built for Bangladesh and Bengali speakers worldwide. It combines Chromium's speed with features designed for local needs: offline tools, Bangla language support, cultural content, privacy tools, and a browser experience that carries the thousand-year heritage of Bengal.

## Features

- **Privacy First**: Built-in ad/tracker blocking, HTTPS-only, DNS-over-HTTPS, encrypted credentials
- **Bengali Language**: Full Bangla UI, voice input, dictionary, cultural content
- **Offline Tools**: 295 utilities, 110 games, books, all work without internet
- **Video Editor**: Professional timeline editing, export to 4K
- **Device Sync**: QR-code pairing between desktop and mobile
