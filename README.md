## What it is

An open-source browser (MIT) with **zero third-party trackers, zero ads, zero telemetry, and zero account required**. Your data is encrypted on-device — we can't read it even if compelled. Ships with **295 offline tools, 110 original games, a cultural archive, and a professional video editor** — all working without an internet connection.

## Privacy & security

- **Zero third-party trackers by default** — EasyList, EasyPrivacy, Disconnect, OISD, urlhaus, nocoin, fingerprint defense, StevenBlack hosts
- **DNS-over-HTTPS default** — Cloudflare, Quad9, NextDNS, AdGuard, Mullvad, local BD providers, and more
- **E2E-encrypted sync** — we cannot read your data even if compelled
- **HTTPS-only default**, third-party cookies blocked, WebRTC leak-protection on by default
- **Ask-every-time** permissions for camera, mic, location, notifications, clipboard
- **Local breach check** — Have I Been Pwned via downloaded hash prefixes; your password never leaves the device

**We do not collect:** browsing history, search queries, URL bar typing, installed extensions, screen/GPU/timezone fingerprints, or IP addresses.
