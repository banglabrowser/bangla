# Bangla Browser

**Privacy-respecting. Bengali-first. Heritage-powered.**

Bangla Browser is a desktop web browser built for Bangladesh and Bengali speakers worldwide. It combines Chromium's speed with features designed for local needs: offline tools, Bangla language support, cultural content, privacy tools, and a browser experience that carries the thousand-year heritage of Bengal.

## Features

- **Privacy First** — Built-in ad/tracker blocking (4 blocklists), HTTPS-only, DNS-over-HTTPS (8 providers), encrypted credentials, WebRTC leak protection, fingerprint resistance.
- **Bengali Language** — Full Bangla UI, voice input, dictionary, cultural content, Letters & Sounds set, books.
- **Offline Tools** — 295 utilities (Calculator, Notes, QR, Hash, Regex, Color Tools, CSS Generator, Markdown Editor, Periodic Table, Medical Calculators, Drawing Pad, Invoice Generator, and more) — all work without internet.
- **Built-in Apps** — Video Editor with Bangla auto-caption, Bird Cloud sync (QR-pair desktop with mobile, encrypted), Notes vault, Passwords vault (safeStorage-encrypted), Bangla Calendar, Reading List, Reader mode, World Clock.
- **110+ Games** — Sports, cultural, board, puzzle, arcade — all offline. Now with universal sound + keyboard support + gamepad bridge.
- **Built-in VPN** — Paid built-in VPN (WireGuard via WireProxy).
- **Voice Search** — Speak Bangla, search instantly.
- **35 Languages** — UI translated to 35 languages.
